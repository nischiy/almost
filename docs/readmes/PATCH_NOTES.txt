PATCH CONTENTS
--------------
- scripts/maintain_structure.py : moves analysis_indicators.py to scripts/, archives _previews/, rewires imports to unified config.
- core/risk.py : RiskManagerCompat with refresh_from_logs(), daily limits, helpers.
- core/live_guard.py : simple hook to check daily limits before opening a trade.

HOW TO APPLY
------------
1) Unzip patch into your project root so that files land under scripts/ and core/.
2) Run:  python scripts/maintain_structure.py
3) Ensure your .env has RISK_MAX_TRADES_DAY (or old RISK_MAX_TRADES_PER_DAY will still be read).
